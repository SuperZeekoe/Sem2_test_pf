Development

During our programming class we’ve gotten really nice tasks and assignments towards this learning outcome. Frank even gave us a head-start on JavaScript instead of getting it according to the curriculum at week 15 or 16.

We initially started with a task to recreate an existing website. The goal was to recreate the landing page at first glance. For more information on this, please click ‘Task 1’.

The second task was to create a random cat image generator. It started with just using JavaScript to click on a button to generate some text. It evolved into generating a cat image at the coordinates of where your mouse would click on the screen. For more information on this, please click ‘Task 2’.

The final task we received was to create a single HTML file, with multiple CSS files. We had to use the same elements for all three designs, and as an added assignment we were asked in trying to create a button, to swap between the separate styles. For more information on this, please click ‘Task 3’.
